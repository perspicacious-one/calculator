# Command Line Calculator

Using Inquirer for input handling. No other dependencies,

### Getting set up

1. Clone the repo using: `git clone`
2. Navigate the the directory: `cd calculator`
3. Install the calculator _make sure you are in the project directory!_: `npm i -g`
4. Run the app with the command: `calculator`

### Under the hood

The app simply uses the javascript `eval()` function to do the math. Nothin to it!
